# Открытый урок по алгебре

Проект: презентация + доска ученика + ИИ-ассистент для темы уравнений высших степеней.

## Структура
- `backend/` FastAPI + алгоритмы многочленов
- `frontend/` Vite + React + Tailwind + shadcn/ui
- `scripts/` PowerShell скрипты

## Быстрый старт
- Dev: `scripts/dev.ps1`
- Prod: `scripts/build.ps1` затем `scripts/run-prod.ps1`
- Tests: `scripts/test.ps1`
